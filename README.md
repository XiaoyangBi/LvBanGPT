# 旅游助手

**功能模块**

- 根据天气查询结果，给外出旅游建议（连接天气预报API）
- 路径规划（飞机，火车）（相关购票app）
- 目的景点推荐（来自小红书等）
- 当地风俗礼仪，餐饮推荐（来自小红书等）
